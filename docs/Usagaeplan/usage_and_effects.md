# 스파이스 흐름 지도 — 활용 방안 및 기대 효과

> 발표자용 정리본 · 2026 서울시 빅데이터 활용 경진대회

---

## 활용 방안

### 1. 자치구청 정책 담당자 — 상권 활성화 예산 배분

**누가 쓰는가**: 강남구·관악구 등 자치구 경제진흥과
**언제 쓰는가**: 분기별 예산 배분, 도시재생 사업 우선순위 도출, 골목상권 활성화 공모 심사

**활용 시나리오**
- GRI 상위 20% 상권을 우선 지원 대상으로 자동 추출
- PolicyCard(R4~R7)를 그대로 결재 문서로 전환 → 보고서 작성 시간 단축
- 자치구 내 백분위 순위로 **구 내 공정 배분** 근거 확보

### 2. 소상공인 지원기관 — 선제 모니터링

**누가 쓰는가**: 서울신용보증재단, 서울산업진흥원(SBA), 소상공인시장진흥공단
**언제 쓰는가**: 보증·대출 심사, 폐업 위험 사전 경보

**활용 시나리오**
- 흐름 단절 신호가 잡힌 상권 점포에 **선제 컨설팅** 투입
- B1·B3 베이스라인이 놓친 187~231개 상권을 추가 관리 대상에 편입
- Module C 시계열 갭 지표로 폐업 임박 점포 조기 식별

### 3. 도시계획·재생 전문가 — 흐름 단절 진단

**누가 쓰는가**: 서울연구원, 도시재생지원센터, 민간 도시계획 컨설턴트
**언제 쓰는가**: 도시재생 활성화 지역 지정, 가로 환경 개선 사업

**활용 시나리오**
- 실 도로 경로 기반 단절 시각화로 **물리적 가로 단절** 위치 식별
- OD 네트워크 중심성 저하 상권을 도시재생 우선 후보로 선정
- 정책 개입 전·후 GRI 변화로 사업 효과 정량 평가

### 4. 예비 창업자·소상공인 — 입지 의사결정

**누가 쓰는가**: 강남·관악 일대 창업 예정자, 자영업자
**언제 쓰는가**: 점포 계약 전, 업종 전환 검토

**활용 시나리오**
- 상권별 **순유입 추세 + GRI 추이**로 입지 안전성 판단
- 신림·압구정 등 사례 상권 비교로 업종 적합도 검토
- CSV 다운로드로 사업계획서에 데이터 첨부

### 5. 학계·연구기관 — 도시경제 연구 데이터셋

**누가 쓰는가**: 도시경제학·공간정보학 연구자, 대학원생
**언제 쓰는가**: 논문 작성, 정책 효과 분석 연구

**활용 시나리오**
- H1·H2·H3 가설 검증 데이터(n=1,565)를 후속 연구 베이스라인으로 활용
- 규칙 기반 정책 추천 엔진을 **재현 가능한 기준 모델**로 인용
- 공개된 검증 코드(scripts/run_validation_all.py)로 타 자치구 비교 연구

### 6. 시민·주민 — 동네 상권 건강 모니터링

**누가 쓰는가**: 강남·관악 거주 시민, 상권 관련 시민단체
**언제 쓰는가**: 동네 상권 변화 추적, 공공 의견 수렴

**활용 시나리오**
- `?hero=1` 시연 모드를 **공공 키오스크·구청 민원실** 디스플레이에 노출
- 자가 동네 상권의 GRI 추이를 통해 임대료 협상·이주 의사결정 참고
- 투명한 검증 결과로 행정 정책에 대한 시민 신뢰 형성

## 기대 효과

### 1. 정책 효율성 향상 — 사전 개입 → 사후 대응 비용 절감

**핵심 수치**: GRI 상위 20% 상권의 다음 분기 폐업률이 **하위 80%의 14.5배**

- 폐업이 발생한 후 긴급 지원하는 사후 대응 모델 → **3~6개월 선행 지표**로 사전 개입
- 폐업 1건당 평균 사회·경제 비용(점포 손실 + 고용 손실 + 임대 공실) 절감 효과 기대
- 흐름 신호 기반으로 **공식 지표가 놓친 187개 상권**까지 정책 사각지대 해소

### 2. 의사결정 시간 단축 — 분석 보고서에서 즉시 결재로

**Before**: 데이터 분석가가 정책 보고서 수기 작성 → **수일 소요**
**After**: PolicyCard 자동 생성 + CSV 다운로드 → **수 초 내 결재 라인 제출**

- 자치구 경제진흥과의 분기 보고 리드타임 대폭 단축
- 동일 시간에 더 많은 상권을 검토 가능 → 정책 커버리지 확대
- 분석가는 단순 보고서 작성에서 **고도 분석·전략 수립**으로 역할 전환

### 3. 데이터 기반 행정 — 직관·관행에서 통계 근거로

**핵심 차별점**: 모든 정책 카드에 **수치 근거 명시** (생성형 AI 미사용 → 환각 없음)

- 정책 의사결정에 "왜 이 상권을 선정했는가"에 대한 정량 근거 첨부
- 감사·평가 시 추적 가능한 의사결정 로그 확보
- 정치적 압력·외부 청탁에 대한 **객관적 방어 논리** 제공

### 4. 행정 투명성 강화 — 검증 결과 공개

**5장의 검증 카드(H1·H2·H3·B1·B3)** 가 발표 화면에 그대로 노출

- "이 지표가 왜 신뢰할 만한가"를 시민·전문가가 직접 검증 가능
- H2 방향 역전 등 **한계까지 공개**하는 정직한 데이터 거버넌스
- 시민 신뢰도 제고 → 행정 정책 수용도 향상

### 5. 확장 가능성 — 강남·관악에서 서울 전역·전국으로

**MVP 구조**가 그대로 확장 가능

- 강남·관악 **1,650 상권 → 서울 25개 자치구 → 광역시·전국**
- 동일 파이프라인(OA-22300, OA-15577 등 공공 API 6종)으로 추가 데이터 수집 자동화
- 자치구별 GRI 가중치 튜닝으로 **지역 특성 반영**

### 6. 사회적 가치 — 젠트리피케이션 대응 + 골목상권 보호

**누가 가장 큰 수혜자인가**: 임대료 급등으로 밀려나는 영세 소상공인

- GRI 기반 조기 경보로 **임대료 인상 폭주 전 정책 개입**
- 프랜차이즈 잠식 모니터링(R5)으로 동네 상권 다양성 보호
- 도시계획 단계에서 흐름 단절 회피 → **걷고 싶은 거리 조성**에 기여

### 7. 기술적·학술적 기여

- 규칙 기반 정책 엔진을 **재현 가능한 오픈 기준 모델**로 공개 (GitHub: Ju-Rak-E/spiceMap)
- 검증 가설 5종(H1·H2·H3·B1·B3) + 통계 근거를 후속 연구 자산으로 축적
- React + MapLibre + Deck.gl + ORS 통합 사례로 **공공 데이터 시각화 레퍼런스** 제공
