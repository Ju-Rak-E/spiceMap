## 1. 주제 구체화

### 1.1 공식 프로젝트명

**스파이스 흐름 지도 (Spice Flow Map)**  
"서울 상권 불균형을 인구·소비 흐름 네트워크로 시각화하는 정책 지원 플랫폼"

### 1.2 핵심 가치 명제

> 기존 상권 분석이 **상태(스냅샷)**을 보여준다면,  
> 스파이스 흐름 지도는 **왜 그 상태가 됐는가(흐름)**를 보여준다.

- 침체 상권을 사후 대응이 아닌 **선제 대응** 대상으로 전환
- 흐름 단절 지점을 클릭 한 번으로 파악 → 정책 제언 카드 자동 생성
- 서울시 정책팀, 자치구 경제과가 **실무에 바로 쓸 수 있는** 보고서 출력

### 1.3 분석 범위 (MVP 기준)

| 구분 | MVP 범위 | 완성본 확장 |
|------|---------|-----------|
| 지역 | 자치구 2곳 시범 (강남구·관악구) | 서울 전역 25개 자치구 |
| 이동 목적 | 출근, 쇼핑 2종 | 전체 목적 (여가·귀가 포함) |
| 시간 단위 | 분기 단위 비교 | 시간대·요일 필터 |
| AI 제언 | 규칙 기반 템플릿 | 스트리밍 LLM 설명 |
| 내보내기 | 위험 상권 CSV | PDF 자동 보고서 |

### 2.2 기능 상세 명세

#### F-01 서울 상권 흐름 지도 (메인 뷰)

- MapLibre GL + Deck.gl 기반 인터랙티브 지도
- **상권 노드**: 위치 마커, 크기 = 순유입량 비례, 색상 = 상권 유형
- **OD 흐름 곡선**: 상위 N개 흐름만 렌더링 (성능 제한)
- **시간대 애니메이션**: 분기별 전환 시 페이드 트랜지션
- 수락 기준: 지도 초기 로딩 3초 이내, 상권 클릭 반응 500ms 이내

#### F-02 상권 유형 분류

| 유형 | 조건 | 색상 |
|------|------|------|
| 흡수형_과열 | 순유입 상위 + 임대료↑ + 프랜차이즈↑ | 빨강 |
| 흡수형_성장 | 순유입↑ + 매출↑ + 임대료 안정 | 주황 |
| 방출형_침체 | 순유출 지속 + 폐업률↑ | 회색 |
| 고립형_단절 | 유입·유출 모두 낮음 + 연결도↓ | 진회색 |
| 안정형 | 변동성 낮음 + 지표 균형 | 초록 |

- 절대값 + 자치구 내 백분위 병행 사용 (규모 편차 보정)
- 주거형 상권 오판 방지: 목적별 흐름 분해 후 분류

#### F-03 상권 클릭 → 상세 패널

우측 슬라이드 패널에 표시:
```
[상권명]  [유형 배지]  [우선순위 점수]
────────────────────────────────────
GRI: 72.4  ↑ 전 분기 대비 +8.2
순유입: +4,200명/일  폐업률: 12.3%
────────────────────────────────────
흐름 단절 원인  [펼치기]
정책 제언 카드  [펼치기]
데이터 출처     [OA-22300] [OA-15572]
```
- 출처 아이콘 클릭 시 공공데이터포털 링크 이동
- 상권 필터 변경 시 선택 상태 유지 (컨텍스트 유지)

#### F-04 분기 타임라인 슬라이더

- 하단 고정 바 형태
- 2024Q1 ~ 최신 분기까지 이동 가능
- 슬라이더 드래그 시 지도·패널 실시간 갱신
- 두 핸들 모드: 시작 분기 + 끝 분기 선택 → F-12 비교 뷰 연동

#### F-05 GRI (젠트리피케이션 위험 지수)

```
GRI =
  0.30 × 순유입 증가율 (자치구 내 z-score)
+ 0.25 × 임대료 상승률 (분기 변화율)
+ 0.20 × 프랜차이즈 비중 증가
+ 0.15 × 매출 급증률
- 0.10 × 독립 소상공인 비중
```
- 0~100 정규화 / 70+ 경보 / 85+ 위험
- 상권 유형별로 임계값 차등 적용

#### F-06 정책 우선순위 점수

```
Priority Score =
  0.35 × 순유출 위험
+ 0.25 × 폐업 위험
+ 0.15 × 연결 단절도
+ 0.15 × 생활권 수요 대비 격차
+ 0.10 × GRI
```
- 80+ : 즉시 개입 (빨강 배지)
- 60~79 : 연내 지원 (주황 배지)
- 40~59 : 모니터링 (노랑 배지)
- 우선순위 80+ 목록을 메인 화면 사이드바에 기본 표시

#### F-08 흐름 단절 구간 시각화

- 별도 토글 레이어 (기본 OFF)
- 단절 엣지: 점선 강조 표시 + 단절 유형 툴팁
- 출력 포맷:
```json
{
  "상권명": "신림 골목상권",
  "단절유형": "주말 쇼핑 유입 부족",
  "상류차단지점": ["서울대입구역 상권", "봉천역 상권"],
  "정책후보": ["보행 유도 사인", "야간 프로그램"]
}
```

#### F-09 AI 정책 제언 카드

- 규칙 기반 (생성형 AI 미사용, 환각 없음)
- 입력: 수치형 지표만 사용
- 출력 고정 포맷:
```
현황: [상권 유형] 상권이며 최근 [N]개 분기 [추세].
근거: 순유출 [X]명/일, 폐업률 [Y]%, 중심성 [추세].
권고1: [구체 행동]
권고2: [구체 행동]
권고3: [구체 행동]
담당: [부서명1], [부서명2]
기대 지표: [측정 가능한 결과]
```
- 데이터 부족 시 "현재 데이터 부족 — 판단 보류" 출력
- "규칙 기반 정책 제언 | 생성형 AI 미사용" 라벨 명시

#### F-10 CSV 내보내기

- 우선순위 80+ 상권 목록
- 컬럼: 상권명, 자치구, 유형, GRI, 우선순위 점수, 추세, 정책 권고 요약
- 서울시 정책팀 → 자치구 경제과 배포 워크플로우 지원

### 3.4 검증 설계 (심사 대응)

| 가설 | 검증 방법 | KPI |
|------|---------|-----|
| H1: 순유입↑ → 매출↑ | 순유입-매출 증감 Pearson 상관 | r ≥ 0.40 목표 |
| H2: 흐름 단절 → 폐업↑ | 침체 예측 상권 vs 실제 폐업 상권 정합도 | 적중률 ≥ 60% |
| H3: 순유입↑+임대료↑+프랜차이즈↑ → 젠트리피케이션 | GRI 고위험 상권의 다음 분기 임대료/프랜차이즈 변화 | 일치도 측정 |

베이스라인 비교:
- B1: 상권변화지표 단독
- B2: 생활인구 단독
- B3: 직전 추세 연장
- **제안 모델**: OD + 생활인구 + 매출 + 점포 + 임대료 + 상권변화지표

| PostgreSQL + PostGIS 로컬 및 공용 인스턴스 구축 | 기본 지도 렌더링 (서울 행정동 경계 레이어) | 행정동-상권 공간 결합 방식 확정 (폴리곤 교차 면적 기준) |
