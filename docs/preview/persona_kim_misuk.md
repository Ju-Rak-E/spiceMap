# 페르소나 — 김미숙 (50대 여성, 반찬가게 창업 희망)

> 작성: 2026-05-12 · 목적: spiceMap 사용자 여정·UX 설계 기준 페르소나
> 연결 문서: `docs/preview/demo_storyline_gangnam_gwanak.md`, `docs/preview/hero_shot_scenario.md`
> 가공의 인물입니다. 실존 인물·상호와 무관.

## 1. 한 줄 요약

**송파 거주, 25년 가정주부 → 동네 반찬가게 보조 2년 → 강남 또는 관악에서 첫 가게를 열려는 53세 여성.**
요리에는 자신 있지만 상권을 보는 눈은 없어, 자녀의 권유로 spiceMap을 처음 켠다.

## 2. 페르소나 카드

| 항목 | 내용 |
|------|------|
| 이름 (가명) | 김미숙 |
| 나이 / 성별 | 53세 / 여성 |
| 거주지 | 송파구 가락동 (강남·관악 모두 30~40분, 중립적 위치) |
| 경력 | 가정주부 25년 → 자녀 둘 독립 → 동네 반찬가게 보조 2년 |
| 가용 자금 | 보증금+권리금 1억~1.5억 + 운영자금 5천만원 (총 2억 내외) |
| 가족 의견 | 남편: "안정적인 데로 해라" / 딸: "엄마 이번엔 데이터 보고 정해" |
| 데이터 리터러시 | 낮음 — 엑셀 셀 정렬은 가능, 통계 용어는 어려움 |
| 디바이스 | 14인치 노트북 + 안경, 모바일보다 데스크탑 환경 선호 |

## 3. 핵심 고민

1. **강남이면 매출은 크지만 임대료가 무섭다** — 권리금 회수까지 버틸 자신이 없음.
2. **관악이면 임대료 여유 있지만 사람이 줄까 봐 무섭다** — 매출 자체가 나오지 않으면 끝.
3. **한 번 들어가면 권리금 회수까지 최소 3년** — 잘못된 선택의 비용이 매우 큼.

## 4. 성공 기준 / 실패 두려움

- **성공**: 월 순이익 350~400만원, 3년 내 권리금 회수, 단골 비중 60%+
- **실패 두려움**: "5천만원 까먹고 빈손으로 나오는 동네 아주머니"가 되는 것.

## 5. 의사결정 행동 패턴

- 결정 전 가족·지인 3명 이상의 의견을 듣는다.
- 숫자는 무서워하지만 **누군가 옆에서 풀어 설명해주면** 따라간다.
- 한 번 결정하면 잘 바꾸지 않는다 → spiceMap은 **결정 전** 단계에서 만나야 가치가 있음.
- "지금 당장 답"보다 **"지금 들어가지 말아야 할 이유"** 를 제시받는 쪽이 더 안심된다.

## 6. spiceMap이 이 페르소나에게 답해야 할 질문

| 질문 | 대응되는 spiceMap 기능 |
|------|----------------------|
| "여기 사람이 진짜 다니나?" | OD 흐름·유입 변화 시계열 |
| "지금은 좋아 보여도 곧 무너지는 거 아냐?" | GRI 위험지수, 흡수형/방출형 분류 |
| "임대료 압박이 더 세질 동네인가?" | 강남 흡수형 과열 신호 |
| "정부가 지원해주는 건 있나?" | R4~R7 정책 추천 카드 |
| "이 분석을 믿어도 되나?" | 검증 보고 (H1·H3·B1·B3) |
| "딸한테 보여주려면?" | CSV 다운로드 |

## 7. 왜 이 페르소나인가

- **거주지 중립성**: 송파는 강남·관악 어느 쪽으로도 편향 없는 출발점.
- **업종 대표성**: 반찬가게는 주거지형 상권의 대표 업종 — 1인·고령가구 비율과 출퇴근 유동인구가 결정적이라 spiceMap의 인구 흐름 분석이 직접 적용됨.
- **자금 규모의 긴장감**: 2억은 강남 진입에 빠듯하고 관악에는 여유 — 의사결정 갈등이 자연스럽게 발생.
- **데이터 비전문가**: UI 친절도·자막의 명료도를 검증하는 기준 사용자.
- **보편적 공감대**: 시청자가 "우리 엄마/이모도 저럴 것 같다"고 느낄 수 있는 캐릭터 — 발표·시연에서 감정 진입점으로 활용 가능.
