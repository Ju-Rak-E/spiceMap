## 4. 결측 영향 분석

### 4.1 2025Q2 / 2025Q3 결측

- **원인 추정**: 서울 열린데이터광장 API (VwsmTrdarSelngQq)에서 해당 분기 미배포. 분기 데이터 공개 시차 또는 데이터 품질 재검토로 인한 지연 가능성.
- **대응**: `collect_commerce_sales --quarter 20252 --quarter 20253` 주기적 재시도. Week 3 중 1회 재확인.

### 4.2 H1 검증에 미치는 영향

| 검증 방식 | 가용 분기 | 결측 영향 |
|----------|----------|----------|
| 동분기 상관 (순유입_t vs 매출_t) | 6 | 영향 없음 |
| 지연 1분기 상관 (순유입_t vs 매출_{t+1}) | 5 (2024Q1~2024Q4, 2025Q1) | 2025Q1→Q2 매칭 불가 |
| GRI 추세 → 매출 변화율 | 5 인접쌍 | 2024Q4→2025Q1만 유효, 2025Q1→2025Q4는 6개월 간격 주의 |

**권고**: H1 1차 검증은 동분기 상관으로 수행, 지연 상관은 2025Q1→Q4를 "2~3분기 지연"으로 별도 처리하거나 제외.
